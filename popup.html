Отслеживаемые функции

Приложение необходимо для демонстрации параметров, которые передаются вызывающимся функциям.

Список функций ограничен. В связи с особенность js-функций, корректная работа основного web-контента — не гарантируется.

Используете приложение только по назначению, после чего отключайте. Примечание: полное отключение — выключить из-под chrome-a
